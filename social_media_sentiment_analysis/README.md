Social Media Sentiment Analysis

This project analyzes emotions and sentiment from social media posts using Python and NLP.  
It cleans the dataset, performs sentiment classification, and generates visualizations.

 Features
 -->Remove null rows and columns
 -->Sentiment analysis using TextBlob
 -->Visualizations:
 -->Sentiment distribution
 -->Sentiment by platform
 -->Sentiment by country
 --> Word cloud of frequent terms
 -->Daily sentiment trend
